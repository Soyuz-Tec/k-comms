# Staffing Plan Draft

## Core roles

- Engineering/architecture lead
- Elixir/OTP backend engineers
- Web client engineers
- Mobile engineers where in scope
- Platform/SRE engineers
- Data/database engineer
- Security engineer or embedded security partner
- QA/test-automation and performance engineer
- Product manager and product designer
- Technical writer/developer experience owner as scale requires

## Illustrative team shape

A production web-first release commonly needs several parallel squads or a cross-functional group covering approximately:

- 4–8 backend/realtime engineers
- 2–5 web/client engineers
- 2–3 platform/SRE/data engineers
- 1–2 test/performance engineers
- Shared security, product, design, and program leadership

This is a planning hypothesis, not a committed estimate. Refine through scope, reuse strategy, client count, compliance, media, and schedule constraints.
