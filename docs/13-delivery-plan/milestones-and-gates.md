# Milestones and Gates

| Gate | Required evidence |
|---|---|
| Architecture baseline | Approved critical ADRs, diagrams, protocol invariants, open-risk disposition |
| Foundation ready | Repeatable staging deployment, auth skeleton, telemetry, safe migrations |
| Messaging ready | Ordering/idempotency/replay tests, load benchmark, tenant isolation |
| Feature complete | End-to-end journeys, contracts, admin/audit/retention, client compatibility |
| Production ready | SLO monitoring, runbooks, restore/DR, security review, capacity and cost approval |
| General availability | Controlled launch, support readiness, error budget, rollback and communications |
