# Release Checklist

- [ ] Scope and risk reviewed
- [ ] Artifact built reproducibly and scanned
- [ ] Contracts remain compatible
- [ ] Migrations rehearsed and bounded
- [ ] Staging smoke/load tests pass
- [ ] Dashboards, alerts, and runbooks are ready
- [ ] Canary and abort thresholds defined
- [ ] Rollback tested
- [ ] On-call and communications owners assigned
- [ ] Post-release verification completed
